# [AMXX] Arena [1v1]

Arena is a mod for CS 1.6 that lets players face each other in a one on one battle. The winner will go up in "level" while the loser will go down.

## Requirements

```
- AMXMODX 1.8.3+
- ReHLDS
- ReGameDLL
- ReAPI
```
